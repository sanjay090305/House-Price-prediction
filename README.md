House Price Prediction using Machine Learning
Project Overview
This project predicts house prices using Machine Learning techniques. A Linear Regression model is trained on housing data to estimate property prices based on various features.

Technologies Used
Python
Google Colab
Pandas
NumPy
Matplotlib
Scikit-learn

Project Workflow
1.Data Collection
2.Data Preprocessing
3.Exploratory Data Analysis
4.Feature Selection
5.Model Training using Linear Regression
6.Model Evaluation
7.House Price Prediction

Results
The model was evaluated using Mean Absolute Error (MAE), Mean Squared Error (MSE), and R² Score.
